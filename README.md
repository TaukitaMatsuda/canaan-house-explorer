# Canaan House Explorer

Интерактивный проводник по Каанан-Хаусу из книги «Гидеон Девятый» (Gideon the Ninth) Тамсин Мьюр.

## Требования

- Node.js 18+
- npm или yarn

## Установка и запуск

```bash
# 1. Распакуйте архив
cd canaan-house-explorer

# 2. Установите зависимости
npm install

# 3. Запустите dev-сервер
npm run dev

# 4. Откройте в браузере
# http://localhost:5173/canaan-house-explorer/
```

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы появятся в папке `dist/`.

## Деплой на GitHub Pages

1. Измените `homepage` в `package.json` на свой репозиторий
2. Выполните:
```bash
npm run deploy
```

## Функционал

- **Интерактивная карта** — исследуйте комнаты Каанан-Хауса
- **Персонажи** — фильтры по Дому, роли, статусу
- **Таймлайн** — хронология событий день за днём
- **Кодекс** — энциклопедия мира Запертой Гробницы
- **Документы** — найденные дневники, письма, записи
- **Поиск** — глобальный поиск по всему контенту
- **Двуязычность** — EN/RU
- **День/Ночь** — переключение атмосферы
